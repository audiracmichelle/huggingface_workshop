Workshop Agenda

1. Navigate the Hugging Face Ecosystem

Explore Spaces, tasks, datasets, and models on the Hub. Read dataset and model cards. Understand Parquet and Safetensors formats. Work with the datasets and transformers libraries.

2. Use Hugging Face MCP with Claude Code

Access Claude through Harvard billing and explore MCP-powered workflows.

3. Run Models in Minutes

Use pipeline() for inference. Compare NVIDIA SegFormer and TCD-SegFormer on a test sample.

4. Build an End-to-End Training Loop

Fine-tune SegFormer on the Restor dataset. Track evaluation metrics.
